# Automatic Text Summarization

This project is an NLP-based automatic text summarization system developed using Python and Flask.

## Features
- TF-IDF summarization
- Cosine similarity
- Flask GUI
- NLP preprocessing

## Technologies Used
- Python
- Flask
- NLTK
- Scikit-learn
- NetworkX

## Run Project

Install requirements:

pip install -r requirements.txt

Run:

python summarizer_gui.py
